# Engineer — нейросеть для инженерных расчётов 🏗️

## 📌 Описание
Нейросеть предсказывает толщину утеплителя для стен по трём параметрам:
- 🌡️ Температура воздуха (°C)
- 🧱 Тип стены (0 — кирпич, 1 — газобетон, 2 — дерево)
- 📊 Класс энергоэффективности (0 — эконом, 1 — стандарт, 2 — пассивный)
